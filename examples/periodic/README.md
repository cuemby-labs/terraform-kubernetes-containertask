# Periodic Task Example

Start periodic task by root moudle.

```bash
# setup infra
$ tf apply -auto-approve \
  -target=kubernetes_namespace_v1.example

# create service
$ tf apply -auto-approve \
  -target=module.normal

$ tf apply -auto-approve \
  -target=module.concurrent

$ tf apply -auto-approve \
  -target=module.suspend
```
